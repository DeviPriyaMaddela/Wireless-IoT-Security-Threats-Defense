## Project Overview
This repository contains an academic assignment analyzing major security threats in Wireless and IoT Networks and the corresponding defense mechanisms. The study focuses on vulnerabilities in wireless communication systems and Internet of Things (IoT) environments.


## Topics Covered
- Eavesdropping and Packet Sniffing  
- Man-in-the-Middle (MITM) Attacks  
- Denial of Service (DoS) Attacks  
- Weak Authentication in IoT Devices  
- Malware and Botnet Attacks  
- Encryption Techniques (WPA3, TLS)  
- Intrusion Detection Systems (IDS)  
- Secure Firmware Updates  
- Network Segmentation  


## Files Included
- LaTeX source files  
- Assignment ZIP file  
- Generated PDF document  


## Objective
The objective of this assignment is to understand security risks in wireless and IoT networks and examine effective countermeasures to ensure confidentiality, integrity, and availability of data.
